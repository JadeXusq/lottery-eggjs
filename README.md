# lotteryEggJs

抽奖服务端接口

## 简介

简单的抽奖h5应用
前端框架：vue3 + vite + vantUI + pinia + vueRouter
后端框架：eggJS + mongodb + jwt

### Development

```bash
npm i

```

```bash
npm start

```
